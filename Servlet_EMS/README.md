# Employee Management System 👨‍💼

## Concepts Used
- Java Servlets
- JDBC
- HTML Forms
- MySQL Database
- MVC Basics

## Description
A basic web application developed using Java Servlets that allows users to add employee details through a web form and store them in a database.

## Features
- Add Employee button
- Employee registration form
- Collect employee details
- Store employee information in MySQL database
- Success message after insertion
- Database connectivity using JDBC

## Technologies Used
- Java
- Servlets
- JDBC
- MySQL
- Apache Tomcat
- HTML

## Project Flow
1. User opens the application.
2. Clicks on the **Add Employee** button.
3. Enters employee details.
4. Clicks **Submit**.
5. Data is stored in the database.
6. Browser displays a success message indicating that one record has been inserted.

## How to Run
1. Import the project into Eclipse.
2. Configure Apache Tomcat Server.
3. Create the required MySQL database and table.
4. Update database credentials in the JDBC code.
5. Start Tomcat Server.
6. Open the application URL in a browser.
7. Add employee details and submit the form.

## Output
- Employee details successfully stored in the database.
- Confirmation message displayed in the browser.
